📊 E-Commerce Analytics Dashboard
About the Project

This project is an Excel-based E-commerce Analytics Dashboard designed to help online stores track key performance metrics, understand customer behavior, and visualize sales trends.

Key Features:
Monthly and yearly sales analysis
Year-over-Year (YoY) growth comparisons
Top-performing products and categories
Customer trends and behavior insights
Interactive dashboard with charts and pivot tables
Data
Dataset: data/ecommerce_data.xlsx (sample dataset included)
Note: To use your own data, ensure it follows the same column structure as the sample file.
How to Use
Open the file Ecommerce_Dashboard.xlsx.
Enable macros if prompted.
Navigate through the dashboard sheets to explore the insights.
Update the raw data sheet to refresh all charts and calculations automatically.
